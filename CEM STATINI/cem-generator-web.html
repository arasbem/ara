Istruzioni

Per generare i file settimanali:

  • Inserisci il tuo nome e cognome
  • Seleziona mese e anno
  • Clicca su "Genera file"
  • Scarica il file ZIP contenente tutti i file generati

I file generati avranno:

  • Il tuo nome e cognome nella cella C1
  • Le date di inizio e fine nella cella I1 e K1
  • Il mese/anno nella cella O1

Nota: Il template Excel (dima.xlsx) è già incluso nell'applicazione.
